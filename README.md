# RAG Chatbot

## Alur Sistem

1. Upload file PDF
2. Ekstraksi teks dari dokumen
3. Chunking (memecah teks menjadi bagian kecil)
4. Embedding (mengubah teks menjadi vektor numerik)
5. Menyimpan data ke database vektor
6. Input pertanyaan dari pengguna
7. Mencari data yang relevan
8. AI menghasilkan jawaban

---

## Teknologi yang Digunakan

- Python 3.10.6
- PyPDF (PDF Loader)
- Streamlit (Web UI)
- Sentence Transformers (Embedding)
- ChromaDB (Vector Database)
- Groq (LLM API)
